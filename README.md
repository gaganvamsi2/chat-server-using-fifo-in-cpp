# chat server using fifo in cpp
 chat server and client programs in cpp

execute these commands in terminal 

```linux commands
g++ -o s server.cpp
g++ -o c1 c1.cpp
g++ -o c2 c2.cpp
g++ -o c3 c3.cpp
```
now open 4 new terminal and execute each command in  different terminal
```linux commands
./s
```
```linux commands
./c1
```
```linux commands
./c2
```
```linux commands
./c3
```
